lsigp <br/>
auqeu <br/>
guqto <br/>
{dana <br/>
hlnct <br/>
leqiy <br/>
tswt} <br/>

### <span id = "f28">28. 凯撒与Vigenère</span>

分析密文：数字**+**小写字母**+**大写字母 <br/>

分成3组看看： <br/>
7  8  4  2  9  2  0 <br/>
g  x  c  d  h  c  c <br/>
B  L  U E  D O N <br/>

猜测**BLUEDON**是一个密文，尝试破解 <br/>

密文对照表 <br>

![images](/images/2017-09-20/ld85.png)

------

7  8  4  2  9  2  0 <br/>
h  I  w h  k  q  p <br/>

凯撒一般都是字母位移加密，但上面有数字，猜测应该是位移值。<br/>

尝试破解得到：**aasfbop** <br/>

格式：**flag{aasfbop}** 提交成功
